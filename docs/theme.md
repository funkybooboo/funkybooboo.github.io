# Design System

**Goal:** Simple, Elegant, Minimal.
**Constraint:** Strict adherence to the 5-color palette.

## Palette

| Color | Hex | Visual Role |
| :--- | :--- | :--- |
| **Eggplant** | `#655560` | **Background**. Deep, muted purple base. |
| **Lavender Blush** | `#FCF7FF` | **Primary Text**. High contrast white/pink. |
| **Silver Sand** | `#C4CAD0` | **Secondary Text**. Muted grey for code and metadata. |
| **Battleship Grey** | `#878C8F` | **Structure**. Borders, footer lines. |
| **Rose Dust** | `#A4969B` | **Accent**. Link underlines, keywords. |

## Typography

*   **Body**: System San-Serif (Light `300`).
*   **Headers**: System Sans-Serif (Bold `700`).
*   **Code**: Monospace.

## UI Elements

*   **Links**: White text with a `Rose Dust` underline. Hover brightens the underline to `White`.
*   **Code Blocks**: Background is a 20% black overlay to darken the Eggplant base without introducing new colors. Text is `Silver Sand`.
*   **Contrast**: Main text achieves AAA accessibility rating against the background.
